## 📋 Sobre o Projeto

O PetLembra é um sistema para clínicas veterinárias e pet shops. Este é o frontend da aplicação, desenvolvido como uma **Single Page Application (SPA)**.

### Características

| Característica | Descrição |
|----------------|-----------|
| 🔄 **SPA** | Navegação sem recarregar a página |
| 📱 **Responsivo** | Funciona em desktop, tablet e mobile |
| 🔌 **RESTful** | Comunicação com API via fetch |
| 📚 **Swagger UI** | Documentação interativa da API |
| 💬 **Notificações** | Toast para feedback visual |
| 🎨 **Design Moderno** | Cores, gradientes e sombras |
